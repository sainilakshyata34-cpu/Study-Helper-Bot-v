Study Helper Bot 📚

An AI-powered Study Helper Bot built using Streamlit and Google's Gemini AI.

Features

- Ask study-related questions
- Generate detailed notes
- Create quizzes with answers
- Generate personalized study plans
- Upload PDF notes and ask questions from the PDF

Technologies Used

- Python
- Streamlit
- Google Gemini API
- PyPDF

Team Members

- Lakshyata 
- Laxmi

Project Description

This project helps students learn more effectively by generating notes, quizzes, study plans, and answers from uploaded study materials using AI.
